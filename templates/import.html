标准模板导入（支持复杂组合题 + 图片）

这是给第三方转换工具使用的标准导入格式，不依赖本系统内部题库备份结构。 组合题使用父题 type=composite + children 数组表示，每个小题都可以独立包含题干、选项、解析和分值。

推荐容器格式为 .zip，内部至少包含一个 template.json 和一个可选的 assets/ 目录。 如果题干、解析或选项里有图片，请把图片放进 assets/，并在 HTML 中使用相对路径如 assets/q1.png。

取消
推荐优先使用 .zip。纯 .json 模板适合无图片场景；带图片时请使用 .zip。

标准模板结构
{
  "format": "family-exam-helper-template",
  "version": 1,
  "subjects": ["数学", "物理"],
  "questions": [
    {
      "subject": "数学",
      "tag": "几何,组合题",
      "type": "composite",
      "difficulty": 2,
      "content": "<p>阅读材料：</p><p><img src=\"assets/figure-1.png\"></p>",
      "explanation": "<p>整组解析</p>",
      "children": [
        {
          "type": "single_choice",
          "difficulty": 2,
          "score": 3,
          "content": "<p>根据上图，选择正确答案。</p>",
          "options": [
            {"label": "A", "content": "选项A"},
            {"label": "B", "content": "选项B"}
          ],
          "answer": "A",
          "explanation": "<p>小题解析，可继续引用 <img src=\"assets/detail.png\"></p>"
        }
      ]
    }
  ]
}
  • format 必须为 family-exam-helper-template
  • questions 中普通题与组合题共用同一入口；组合题必须用 children 保存小题
  • 图片引用建议只使用 assets/文件名 或 ./assets/文件名
  • 选项使用数组格式：[{"label":"A","content":"..."}, ...]
  • 小题不需要单独写 subject 和 tag，默认继承父题语义

Excel 格式说明

第一行为表头，从第二行开始为数据。格式如下：

A 列 B 列 C 列 D 列 E 列 F-I 列 J 列 K 列
科目 标签 题型 难度 题干 选项 A-D 正确答案 解析
数学 函数 single_choice 2 求f(x)=x²的导数 2x / x² / 2 / x A 幂函数求导公式
计算机 循环 multi_choice 3 以下哪些是Python循环 for / while / if / switch A,B if是条件判断

题型值： single_choice（单选）、 multi_choice（多选）、 true_false（判断）、 fill_blank（填空）

难度值： 1（简单）、2（中等）、3（困难）

多选答案： 用逗号分隔，如 A,B,C

填空多个答案： 科目不存在时会自动创建

⚠️ Excel 导入仅支持纯文本简单题，不支持带图片的题目，也不适合导入组合题。含图片或复杂组合题请使用上方的标准模板导入。
