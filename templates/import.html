Excel 格式说明

第一行为表头，从第二行开始为数据。格式如下：

A 列 B 列 C 列 D 列 E 列 F-I 列 J 列 K 列
科目 标签 题型 难度 题干 选项 A-D 正确答案 解析
数学 函数 single_choice 2 求f(x)=x²的导数 2x / x² / 2 / x A 幂函数求导公式
计算机 循环 multi_choice 3 以下哪些是Python循环 for / while / if / switch A,B if是条件判断

题型值： single_choice（单选）、 multi_choice（多选）、 true_false（判断）、 fill_blank（填空）

难度值： 1（简单）、2（中等）、3（困难）

多选答案： 用逗号分隔，如 A,B,C

填空多个答案： 科目不存在时会自动创建

⚠️ Excel 导入仅支持纯文本题目。含公式、代码、图片的题目请在网页上编辑添加。
